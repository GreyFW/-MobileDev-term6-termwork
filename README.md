# WorkUP - Трекер тренировок

Семестровая работа по дисциплине "Разработка мобильных приложений". 
Приложение позволяет записывать время тренировки, данные о подходах и вести заметки.

## Выполненные критерии:
* **Чистая архитектура (Clean Architecture):** Проект разбит на 3 модуля (app, data, domain). DI реализован через Hilt (`DataModule.kt`, `MainViewModel.kt`).
* **Фоновые задачи:** Использован `WorkManager` и `BroadcastReceiver` (`BootReceiver.kt`) для фоновой проверки прогресса.
* **Анимации в Compose:** Реализована анимация изменения цвета, вращения и масштаба кнопки сохранения `SaveStreakButton.kt` с помощью `updateTransition`.
* **XML + Compose:** Загрузочный экран `SplashActivity` использует `activity_splash.xml` со встроенным `ComposeView`.
* **Конфигурация сборок:** В `build.gradle.kts` настроены `productFlavors` (dev/prod) и активирован R8/ProGuard для release сборки.
* **Хранение данных:** База данных реализована через `Room Database` (слой data).
